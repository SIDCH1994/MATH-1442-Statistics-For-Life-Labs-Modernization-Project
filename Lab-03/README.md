## ğŸ“ Dataset

Students analyze the **Pennsylvania High School Seniors dataset**, collected as part of the *U.S. Census at School* project.

### Dataset Characteristics

- ~338 observations and ~30 variables
- Mix of **numerical** and **categorical** variables
- Domains include:
  - Physical characteristics (e.g., height, arm span)
  - Demographics
  - Behavior and lifestyle measures
  - Communication and technology usage

This dataset is well-suited for descriptive analysis, group comparisons, and discussion of variability and outliers.

## ğŸ”„ Conceptual Workflow

Rather than focusing on software mechanics, this lab emphasizes **statistical reasoning**:

1. **Context & Variables**
   - Identify numerical vs. categorical variables
2. **Numerical Summaries**
   - Describe center and spread using summary statistics
3. **Visualization**
   - Examine distribution shape using histograms and box plots
4. **Comparison**
   - Compare distributions across groups using side-by-side box plots
5. **Relative Standing**
   - Use Z-scores to interpret individual values within a distribution
6. **Interpretation**
   - Draw conclusions grounded in context and variability
