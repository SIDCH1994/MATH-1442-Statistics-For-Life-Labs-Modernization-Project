## üîç Lab Overview

Lab 02 focuses on exploratory data analysis (EDA) through tables and graphical summaries. Building on the data-handling foundations introduced in Lab 01, this lab helps students understand why graphs matter, how different visualizations encode information, and when to use specific plots based on data type and analytical goals.

Using Jamovi and selected visualization modules, students learn to transform raw data into frequency tables, histograms, and categorical plots, and to compare samples versus populations visually.

## üéØ Learning Objectives

By the end of this lab, students are able to:

- Create frequency and relative frequency distributions

- Transform continuous numeric variables into categorical (binned) variables

- Construct and interpret histograms and relative histograms

- Compare distributions across groups using categorical variables

- Visually compare sample and population distributions

- Select appropriate visualization tools for different data types

- Communicate insights using clear graphical summaries


## üß† Key Statistical Concepts
**Tabular Summaries**

- Frequency distributions

- Relative frequency distributions

**Graphical Summaries**

- Histograms (count vs density)

- Relative histograms

- Bar plots

- Pie charts

- Dot plots

**Distributional Features**

- Shape (skewness, modality)

- Center (mean, typical values)

- Spread (range, variability)

- Representativeness of samples

**Sampling & Comparison**

- Visual comparison of population vs random sample

- Impact of sampling on distributional appearance

## üîÅ Lab Workflow (High-Level)

1. **Enable Reproducible Analysis**

  - Activate Jamovi‚Äôs syntax mode to view the underlying R code

2. **Random Sampling**

  - Generate a random sample using Jamovi‚Äôs filtering tools

  - Toggle between sample and population views

3. **Frequency Distributions**

  - Transform continuous variables into categorical bins

  - Create frequency and relative frequency tables

4. **Graphical Exploration**

  - Plot histograms and relative histograms

  - Compare distributions across categorical groups

5. **Distribution Comparison**

  - Visually compare the sample vs. the population

  - Assess similarity in shape, center, and spread

6. **Alternative Visualizations**

  - Construct bar plots, pie charts, and dot plots

  - Evaluate strengths and limitations of each plot type
